# Sub-processors

The following third-party services are used to provide the Platform services:

- LiteLLM (local compute, model routing)
- OpenRouter (cloud inference, optional fallback)
- Cloudflare (CDN, reverse proxy, DDoS protection)
- Vast.ai (optional GPU compute for heavy workloads)
- Anthropic Claude (optional LLM for complex reasoning tasks)
- GitHub Models (Azure-backed LLM inference, optional)
- OpenAI (optional, subject to API availability)
- Google Gemini (optional, subject to API availability)
